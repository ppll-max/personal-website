# 个人品牌网站 — 设计规格书

**日期**: 2026-06-15  
**项目**: 工商管理大二学生个人网站  
**目标**: 提升实习与管培生申请竞争力  

---

## 一、项目定位

**简历/名片型**个人网站，单页设计。面向企业 HR 和管培生面试官，在 10-30 秒内建立「稳重可靠、有成长潜力」的第一印象。

**用户画像**: 工商管理大二学生，目标管培生，目前经历偏少，通过展示成长轨迹和学习潜力来弥补经验不足。

---

## 二、视觉设计

### 配色方案（灰蓝）

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#2c5282` | 导航栏、主标题、重点元素 |
| 辅色 | `#5a9bd5` | 边框、图标、技能进度条 |
| 底色 | `#edf2f7` | 页面背景 |
| 卡片 | `#ffffff` | 内容卡片背景 |
| 文字 | `#1a202c` / `#4a5568` / `#718096` | 三级文字层次 |
| 点缀 | `#a0c4e8` | 导航栏副文字 |

### 字体

- 中文：思源黑体 Noto Sans SC（Web Font）→ 系统默认字体降级
- 英文/数字：system-ui, -apple-system, Segoe UI
- 字号层级：22px（姓名）→ 16px（模块标题）→ 13px（正文）→ 11px（辅助说明）

### 布局

- 最大宽度 720px，居中单列
- 头部导航固定 + 内容区卡片网格（2列）
- 移动端自动降级为单列

## 三、页面结构（从上到下）

| # | 模块 | 内容 | 视觉效果 |
|---|------|------|----------|
| 1 | **导航栏** | 姓名 + 定位标签 + 锚点导航 | 主色背景，固定顶部 |
| 2 | **个人陈述** | 照片（圆形）+ 姓名 + 一句话定位 + 2-3句职业目标 | 左图右文，分隔线 |
| 3 | **教育背景 + 核心能力** | 双列卡片网格 | 左侧蓝色边线标识 |
| 4 | **课程项目** | 2-3 个课程/社团项目卡片 | 浅灰底 + 白色内卡 |
| 5 | **成长轨迹** | 竖排时间线（已完成 + 进行中） | 节点逐个点亮动画 |
| 6 | **联系方式** | 邮箱、电话、LinkedIn | 居中横排图标 |

---

## 四、动态交互效果

| 效果 | 触发条件 | 实现方式 |
|------|----------|----------|
| 滚动渐入 | 卡片进入视口 | CSS `@keyframes` + Intersection Observer |
| 悬浮微动 | 鼠标 hover 卡片 | `transform: translateY(-2px)` + `box-shadow` 加深 |
| 时间线动画 | 页面加载后 | 节点按时间顺序依次点亮（延迟 stagger） |
| 导航高亮 | 滚动到对应区域 | Intersection Observer 监听 + 导航栏样式切换 |
| 平滑滚动 | 点击导航锚点 | `scroll-behavior: smooth` |

---

## 五、技术选型

| 层面 | 选择 | 理由 |
|------|------|------|
| 框架 | **纯 HTML/CSS/JS** | 单页简历无需框架，加载快、维护简单 |
| 部署 | **Vercel / GitHub Pages** | 免费、自动 HTTPS、自定义域名支持 |
| 图片 | 本地静态资源 | 头像放在 `/images` 目录 |
| 字体 | Google Fonts CDN | 加载思源黑体 |
| 响应式 | CSS Grid + Media Queries | 自适应桌面/平板/手机 |

## 七、非功能需求

- 首屏加载 < 2 秒（无外部依赖，纯静态）
- 移动端可读性 ≥ 桌面端
- 语义化 HTML，支持屏幕阅读器
- 所有动画尊重 `prefers-reduced-motion`
